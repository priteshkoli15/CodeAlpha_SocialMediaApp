# Mini Social Media Platform

A full-stack mini social media web application built using Node.js and Express.js.

## 🚀 Features

- User Registration & Login
- User Profiles
- Create Posts
- Like System
- Comment System
- Follow User System
- Responsive Dark UI
- JSON-based Data Storage

---

## 🛠 Technologies Used

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
